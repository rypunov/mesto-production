# Mesto

Приложение для размещения фотографий в общем фотоальбоме.

## 🌐 Демо

https://rypunov.github.io/mesto-production/

## 🛠 Технологии

- HTML5, CSS3, JavaScript
- Vite
- GitHub Actions
- REST API
- Адаптивная верстка, Flexbox, Grid
- Валидация форм (кастомная)

## 📱 Функциональность

- Редактирование профиля
- Обновление аватара
- Добавление/удаление карточек
- Лайки и счётчик лайков
- Открытие изображений
- Валидация форм с динамическими сообщениями
- Статистика пользователей

## 🚀 Запуск проекта

```bash
# Установка зависимостей
npm install

# Запуск dev-сервера
npm run dev

# Сборка проекта
npm run build
